## Features

## Getting Started

### Frontend

```bash
# install dependencies
npm install

# run the Vite dev server
npm run dev

# launch the Electron app (after building or during development)
npm start
```

### Backend

```bash
cd backend
npm install

# start the API server
npm start
# or run with nodemon
npm run dev
```
